# 🚀 Show Your Potential

A simple and professional survey form designed to understand students' and professionals' technical skills, programming knowledge, academic background, and career interests.

This initiative was created as part of a learning journey to improve form-building, data collection, and networking skills.

## 🌐 Live Form

https://forms.gle/TB5NvGAGcHMiqoAy8

## ✨ Features

* 👤 Full Name Collection
* 📧 Email Address
* 📱 Mobile Number
* 🎓 Branch Selection
* 📚 Academic Year Selection
* 💻 Programming Skills
* ⚙️ Technical Skills
* 🎯 Career Interests
* 💬 Thoughts & Feedback Section
* 📱 Mobile Friendly
* 🔒 Safe & Educational Purpose

## 📋 Information Collected

| Field              | Description           |
| ------------------ | --------------------- |
| Full Name          | Participant's Name    |
| Email ID           | Contact Email         |
| Mobile Number      | Contact Number        |
| Branch             | Academic Branch       |
| Year               | Current Academic Year |
| Programming Skills | Languages Known       |
| Technical Skills   | Tools & Technologies  |
| Career Interest    | Preferred Career Path |
| Feedback           | Additional Thoughts   |
